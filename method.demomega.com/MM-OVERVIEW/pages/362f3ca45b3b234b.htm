BUILDING BLOCK - Relational Entity

DescriptionAbstract Concept
A  Relational Entity is a relational data structure that is either a Table or a Table View.
A  Relational Entity is accessible by means of a primary key, and if necessary foreign keys; it is described by an ordered sequence of Columns.
