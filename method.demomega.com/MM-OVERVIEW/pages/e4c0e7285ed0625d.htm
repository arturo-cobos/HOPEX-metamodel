BUILDING BLOCK - Application System Scenario

Description
An  Application System Scenario is a story involving the components of an Application System (Applications), interacting in order to produce Information Outcomes.
This includes:
1) A course of events represented by  Application Flows depicting the story towards the delivery of expected Information Outcomes.
2) Interacting Applications who participate to the story.
