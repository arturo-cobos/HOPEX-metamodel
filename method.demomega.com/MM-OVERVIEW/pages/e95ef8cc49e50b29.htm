DEFINITION - Workflow Subject

Description
A  Workflow Subject is the object to which a  Workflow Instance applies.
The  Workflow Instance manages the sequence of operations expected to be performed on the  Workflow Subject.
Example: object validation.
The  Workflow Instance can manage the various states of the  Workflow Subject. For instance, as far as validation is concerned, an object can be "validated" or invalidated".
