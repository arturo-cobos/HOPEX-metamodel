BUILDING BLOCK - Appraisal Type

DescriptionAbstract Concept
An  Appraisal Type refers to a kind of Appraisal method and technique used to evaluate the worth, effectiveness, compliance, or quality of enterprise Functional Assets.
 Appraisal Types define a set of Appraisal Options which are the possible values that will result from an Appraisal activity.
For instance, when making decisions about invesment on an enterprise asset, the possible options can be: eliminate, invest, migrate.
Assessment Type and Resolution Type are the two appraisal techniques provided by HOPEX.
