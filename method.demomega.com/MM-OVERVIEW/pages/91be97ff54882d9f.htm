BUILDING BLOCK - Project

Description
A  Project is a course of action that is being executed or has been selected for execution. An organization's Initiatives represent the choices the organization has made about how to pursue the change that allows it to achieve its objectives.

A project has a purpose with respect to the enterprise capabilities (as defined in a capability map); it may:
- Deliver the means to acquire a new capability (new);
- Extend the coverage of an existing capability (improvement);
- Decommission or limit the coverage of an existing capability (rationalization).
