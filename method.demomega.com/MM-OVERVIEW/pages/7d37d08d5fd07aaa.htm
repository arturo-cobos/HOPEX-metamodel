BUILDING BLOCK - Physical Data Object

DescriptionAbstract Concept
A  Physical Data Object  represents the physical structure of any kind of data that can be memorized by a Software System.
A  Physical Data Object is either a Physical Data Entity or a Physical Data Property.
Only Physical Data Entitys can have an identity can by be referenced by Physical Relationship. Physical Data Property only handle raw data.

Last Update: HOPEX V2
