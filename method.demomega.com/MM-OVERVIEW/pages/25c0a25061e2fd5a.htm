BUILDING BLOCK - Software System Environment

DescriptionAbstract Concept
A  Software System Environment is an operating context in which a Software System defines its interactions with its partners (Partner System) in the form of API connections ( Software Interaction Channel).
It also represents End Users that interact with the system.
