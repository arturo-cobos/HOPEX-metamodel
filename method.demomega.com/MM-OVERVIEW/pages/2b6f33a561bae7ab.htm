BUILDING BLOCK - Resource Agent Environment

DescriptionAbstract Concept
A  Resource Agent Environment is an operating context which defines the interactions ( Business Service Channel) of a Resource Agent with its partners (Partner Resource Agent).
VersionUnder construction
Super Types  Business Operating Environment    Resource Ecosystem  
Definition Domain  EA Pattern - Business Resource Operating Pattern
The EA Pattern - Business Resource Operating Pattern domain defines concepts and relationships used to model how physical resources (Resource Operating Assets) operates/functions in Resource Ecosystems to deliver Business Outcomes.
