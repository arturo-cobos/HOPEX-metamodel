BUILDING BLOCK - Environmental Value

Description
An  Environmental Value is a specific value of an Environmental Property.
 Environmental Values are used as constraints on Architecture Blocks in the context of non-functional requirement analysis and Value Proposition analysis.
Examples:
- Muddy, frozen ground, deep snow (Terrain Type)
- Broad daylight, dusk, moonlit  (Light condition)
