DEFINITION - Assessment Template

DescriptionThe assessment template defines the assessment scope, the type of questionnaire to be used, and if required, the aggregation schemas to be applied for interpretation of global results. It is used as a model for definition of one or several campaigns or assessment sessions. The assessment template can also supply deployment queries that can be activated from one session to another to better determine the scope of objects to be assessed, the context and even the respondents.
