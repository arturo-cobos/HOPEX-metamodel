DEFINITION - Scenario of IT Service Flows

DescriptionA scenario of IT service flows presents the flow exchanged between the IT services or the micro-services used by this IT service. A scenario can represent a specific use case of this IT service or, globally, the set of all flows exchanged inside this IT service.
