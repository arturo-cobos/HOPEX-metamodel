BUILDING BLOCK - Data Set Snapshot

Description
A  Data Set Snapshot is status of a Meta Dataset a the time of data discovery. It provides factual information on the Meta Dataset such as Number of Unique Values, Number of Entities, Number of Completed Records, etc.
 Data Set Snapshots are used for quality assessments in data governance.
