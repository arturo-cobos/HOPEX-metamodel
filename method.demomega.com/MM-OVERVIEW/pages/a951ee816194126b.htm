CONCEPT DESCRIPTIONS

Concrete Concepts
ConceptDescriptionHOPEX Metaclass
  Computer Device
A  Computer Device is a Computing System which provides services directly to the end-user.
 Computer Devices can host data stores and run Application code.
Examples: workstation, laptop computer, smartphone.
   IT Device
  Computer Network
A  Computer Network is an assembly of Network Devices (e.g. routers, switches, firewalls) that enables communications between Computational Aspects (e.g. Computer Servers).
A  Computer Network may breakdown into sub-networks.
    References:
  •  AWS - What is Computer Networking?
  •  UCF - Glossary - Network
 		  IT Network
  Computer Server
A  Computer Server is a Computing System which provides processing, storage, and network connectivity services to other Computing Systems on a Computer Network.
As a Computational Aspect, it can host data and run Application code..
    References:
  •  UCF Glossary - Server
 		  IT Server
  Facility
A  Facility is a System of Resources used to model factories, buildings, or outdoor constructions that provides a particular Business Capability or is used for a particular industry.
Examples:
- Data Center
- Factory
- Outlet
    References:
  •  OMG - UAF - Facility
  •  OpenGroup - ArchiMate - Facility
  •  UCF Glossary - Facility
  •  WordNet - Facility
 		  Facility
  Firewall A  Firewall is a network security device that monitors and controls incoming and outgoing network traffic based on predetermined security rules. Its primary function is to prevent unauthorized access to or from a private network by blocking or allowing network traffic.   Firewall
  Hardware System
A  Hardware System is a man made tangible artifact which exposes Hardware Capability(ies) and can produce Physical Outcomes.
A  Hardware System performs System Processes and participates to System Processes or to Business Processes.
A  Hardware System can embed Computational Aspects. Together with its embedded Computational Aspects, a  Hardware System can also deliver Information Outcomes.
A  Hardware System may be based on a set of Hardware Technology(ies).
Examples:
- Connected Drone with Online Payment App.
- 3D printer.
- Automated Guided Vehicles (AGVs)
- Connected fridge providing ordering Functionality and of course a freezing Hardware Capability.
- Production equipment in an assembly line (metal forging machine)
- Car
    References:
  •  OMG - UAF - System
  •  OpenGroup - OAA - Hardware
 		  Hardware
  IT Infrastructure An  IT Infrastructure is a configuration of Computational Aspects and of Computer Networks that  enable the operation the Business Systems of an enterprise.
    References:
  •  UCF - Glossary - IT Infrastructure
 		  IT Infrastructure
  Network Device (Generic) A  Network Device (Generic) is a  Network Device used to represent Network Devices other than those delivered by HOPEX.   Network Device (Generic)
  Resource Configuration
A Resource Configuration is a generic System of Resources that represensed System of Resources that are not Facilitys.(Data centers, Factories, Outlets).
Examples:
- Configured Nuclear power plant (with operating teams), 
- Configured aircraft carrier (with crew), Configured rescue ship (with rescue team).
When specific characteristics need to be considered for a given type of  Resource Configuration, it is recommended to create a new sub-type of System of Resources to host these characteristics.
   Resource Configuration
  Router A  Router is a Network Device that connects multiple Computer Networks together and directs data traffic between them.   Router
  Switch
A  Switch is a Network Device used to connect multiple devices within a Computer Network, enabling them to communicate with each other.
A  Switch operates at the data link layer (Layer 2) of the OSI model, and it uses MAC (Media Access Control) addresses to forward data between Networking Systems.
   Switch
  System of System Environment A  System of System Environment is an operating context which defines the interactions ( Business Service Channel) of a System of Systems with its partners (Partner Resource Architecture).
    References:
  •  OMG - UAF - ResourceArchitecture
 		  Resource Architecture Environment
  System of Systems
A  System of Systems is a Macro Capability Configuration that combine several System of Resources to deliver a common high level Business Capability.
Examples:
- Air traffic control system
- Smart city system
- Health Care System
- Search and Resue system (SAR)
- Naval squadron consisting of an aircraft carrier, its escort ships and its C5ISR capabilities.
    References:
  •  OMG - UAF - CapabilityConfiguration
 		  Resource Architecture
  Wifi Hotspot A  Wifi Hotspot is Network Device that allows multiple devices to connect to a Wi-Fi network.   Wifi Hotspot
Abstract Concepts
ConceptDescriptionHOPEX Metaclass
  Capability Configuration
A  Capability Configuration is a Resource Agent which combines Human Resources and Business Systems along with networking capabilities in order to provide Business Capability(ies).
There are two levels of granularity of  Capability Configurations: System of Systems and System of Resources.
    References:
  •  OMG - UAF - CapabilityConfiguration
  •  OpenGroup - TOGAF - Definition - Business System
 		  Class of Enterprise System of Resource Internal Structure
  Computing System
 Computing System is a physical or virtual computer that can host and run software code.
Together with their Deployable Application Packages, they provide Information Outcomes.
Examples: Computer Device, Computer Server.
  Configured Technology System A  Configured Technology System a Technology System that is a combination of Networking Systems and Computational Aspects.
  Human Resource A  Human Resource is a Resource Agent  used to describe both the Organizational Positions of people who work for an organization and the type of department (Org-Unit) responsible for running business operations.
    References:
  •  Humanresourcesedu.org - Human Resource
  •  OMG - UAF - OrganizationalResource
 		  Class of Organizational Agent
  IT Infrastructure System
An  IT Infrastructure System is an assembly of  Computational Aspects and Network Devices required to to run Business Systems and Business Operations.
An  IT Infrastructure System is either an IT Infrastructure or an Computer Network.
    References:
  •  OpenGroup - OAA - Software Defined Infrastructure
  •  UCF - Glossary - IT Infrastructure
 		  IT Infrastructure
  Network Device
A  Network Device is any piece of physical or virtual hardware that is used to connect Computational Aspects (Computer Devices, Computer Servers) and other devices to a Computer Network.
 Network Devices enable the transmission and exchange of data between devices in the network through Network Interfaces.
Examples: Switch, Router.
   IT Technical Device
  Networking System A  Networking System is a Technology System that provides networking capabilities offered through Network Interfaces.
  System of Resources
A  System of Resources is a Mezzo Capability Configuration that combines Human Resources, Hardware Systems and Software Systems to provide a Business Capability.
Examples:
- Factory (with workers),
- Data center,
- Hospital,
- Configured aircraft carrier (with crew),
- Configured rescue ship (with rescue team).
    References:
  •  DAU - Glossary - System of Systems
  •  Systems_of_Systems_(SoS)
  •  Wikipedia - System of systems
 		  System of Resource
