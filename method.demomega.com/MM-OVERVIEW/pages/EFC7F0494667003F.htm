DEFINITION - SolMan Project

Description
SolMan Project are used to evaluate and implement SAP solutions.
A  SolMan Project defines the set of SAP Transaction used in your SAP system.
Those SAP Transaction are structured in a hierarchy of SolMan Scenario, SolMan Process and SolMan ProcessStep.
The creation of a SolMan Project in MEGA is not possible : the creation is restricted via the MEGA desktop interface.
To start the definition of a SolMan Project in MEGA, first create the Project in Solution Manager, then import it in MEGA.
Once the Project imported in MEGA, it is not possible to modify its properties values : modifications have to be done in Solution Manager, then imported in MEGA.
The System Landscape definition must be specfied in Solution Manager : System Landscape creation and definition is also restricted in MEGA.
VersionAvailable since: MEGA 2007 Beta2
