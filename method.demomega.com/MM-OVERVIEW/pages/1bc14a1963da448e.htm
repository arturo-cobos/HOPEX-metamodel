BUILDING BLOCK - Measurable Property

Description
A  Measurable Property is a concrete Measurable Property (Abstract) that can be applied to all Architecture Block (Type)s.
For instance, "Cost" can be applied to all Architecture Blocks, while "weight" likely applies only to Hardware Systems.
