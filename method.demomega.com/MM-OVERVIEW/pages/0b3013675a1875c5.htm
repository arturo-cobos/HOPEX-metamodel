BUILDING BLOCK - Software Technology Capability Map

Description
A  Software Technology Capability Map is an assembly of Software Technology Capability(ies) and their dependencies that, together, provide a Software Technology Capability scope for an Enterprise Porfolio Stage.
VersionAvailable since: HOPEX V2
Last Update: HOPEX V2

Definition Domain  Software Technology ArcDevOps
The Software Technology ArcDevOps domain describes a set of concepts used in practices that combines Software Technology System development (Dev) and Software Technology System operations (Ops).
