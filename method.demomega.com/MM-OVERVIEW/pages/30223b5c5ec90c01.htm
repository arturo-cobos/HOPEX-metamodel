DEFINITION - Outcome

DescriptionAbstract Concept
An  Outcome is an Event that indicates the availability of an expected change in the state of a Value Added Object brought about by a specific Behavior,
An  Outcome is a connection point between a producer Behavior and a consumer Behavior which benefits from the produced Value Added Object.
In HOPEX, the most common  Outcome is exemplified by the Information Outcome which refers to the production or consumption of a specific Information Object such as an order received or a  payment rejected.
