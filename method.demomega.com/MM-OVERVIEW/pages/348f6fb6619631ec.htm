Source Entity  Class   Class of Object
Abstract
A  Class is collection of Entitys which it classifies according to one or more criteria.
In knowledge representation, a  Class can be defined either by extension (specifying members), or by intension (specifying criteria), using what is called in some ontology languages like OWL.
Not two  Classes have the same membership. However, a distinction must be made between a  Class having members, and those members being known, so within an information system the members recorded may change over time, even though the true membership does not change.
Note: The HOPEX Foundation is a higher-order ontology, so  Classes may have members that are also  Classes (there are  Classes of  Classes).
