DEFINITION - Project Domain

DescriptionA project is defined upon a given domain (e.g. Business, Information Technology)

A domain gathers all the projects managed from the same perspective in terms of portfolio management decisions. Project Portfolios are therefore also attached to Project Domains; by default, one portfolio of each portfolio type is created when creating a new domain.
