BUILDING BLOCK - Operating Context

DescriptionAbstract Concept
An  Operating Context is a perimeter of Architecture Block (Type) that forms the scope of an Enduring Initiative. It can be a set of Capability(ies) that define the purpose of operating Agents or the Operating Eco-System in which these Agents deliver these Capability(ies).
VersionUnder construction
Super Types  Aggregate Block (Type)    Architecture Block (Type)  
Definition Domain  Operating Semantic
The Operating Semantic domain is used to model how a system (Agent) operates/functions (Behavior) in its Operating Eco-System to produce Outcomes that benefit (value) to other Agents or affect (have an effect upon) these Agents.
