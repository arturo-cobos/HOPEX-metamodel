BUILDING BLOCK - Logical Data Entity

Description
A  Logical Data Entity is a logical structure of a Data Entity. As any Data Entity, it has an independent existence and can be uniquely identified.
A  Logical Data Entity is characterized by Logical Relationships it has with other  Logical Data Entity(ies) and by its Attributes.
