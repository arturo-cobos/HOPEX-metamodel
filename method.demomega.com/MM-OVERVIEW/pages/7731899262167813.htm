Target Entity  Relational Data Asset[0..*] Abstract
A  Relational Data Asset is a Data Asset used for description of relational data consumed and produced by Software Systems.
 Relational Data Assets range from Relational Schemas to Column Types.
They are defined in Relational Dictionary(ies) and managed in Data Catalogs.
