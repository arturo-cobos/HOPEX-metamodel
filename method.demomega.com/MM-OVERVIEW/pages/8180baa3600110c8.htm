BUILDING BLOCK - MetaData Object

DescriptionAbstract Concept
A  MetaData Object is Physical Data Object that represents an element of a physical schema as it will be stored in relational or non relational DBMS (NoSQL, triple store, document store, flat files, etc.).
A  MetaData Object is either a Meta Dataset or a MetaData Field.
