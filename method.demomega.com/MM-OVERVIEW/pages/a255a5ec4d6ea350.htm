CORE METACLASSES

  Application
  •   Application
  • Agent

An application is a software component that can be deployed and provides users with a set of functionalities.
  IT Service
  •   IT Service
  • Agent
  • Physical Software Resource
An IT service is a software component of an application, that can’t be deployed alone and that realizes a sub-set of the functionalities of this application either for end users of this application or inside the application (or another application). This includes batch programs.
  System Process
  •   System Process
  • Content Repository Object
  • MetaModel-Overview - Exclusion
A system process is the executable representation of a process. A System Process is formally described by the workflow Events, the Tasks to be carried out during the processing, the sequence of the Tasks and the information exchanged between the participants.
In the BPMN notation, the system process represents a sub-process from the system point of view.
